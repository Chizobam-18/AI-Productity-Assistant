# AI-Productivity-Assistant (Smart Desk Copilot)

An AI-powered workplace productivity platform built for the CAPACITI AI Skill Accelerator Programme. This application is designed to automate administrative blocks, streamline corporate communications, and optimize daily scheduling.

## 🚀 Live Application URL
Click the link below to access and test the live application:

---

## 🛠️ Key Features Included
* **Smart Email Generator:** Instantly converts basic context into polished, tone-adjusted professional email drafts.
* **Meeting Notes Summarizer:** Distills raw, messy meeting notes into highly structured summaries and clear actionable targets.
* **AI Task Planner:** Reorganizes messy task lists into prioritized chronological time blocks on a clean calendar layout.

## 🧠 Applied Prompt Engineering & Skills
* **Context-Aware Prompting:** System triggers explicit persona rules ensuring clear corporate structures for all outputs.
* **Ethical AI Safety Banners:** Integrated clear warning modals to prevent the submission of sensitive data and to remind users to verify AI outputs for accuracy.
* **Rapid UI Orchestration:** Prototyped and cloud-deployed utilizing React, Tailwind CSS, and Lovable.ai framework engines.
